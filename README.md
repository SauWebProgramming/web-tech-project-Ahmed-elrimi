# Personal Portfolio SPA

A modern **Single Page Application (SPA)** portfolio built using **HTML, CSS, and Vanilla JavaScript**.
The project demonstrates client-side routing, dynamic content rendering, responsive design, and clean UI engineering practices.

 **Live Demo:**  
https://sauwebprogramming.github.io/web-tech-project-Ahmed-elrimi/

---

## Project Overview

This portfolio website is designed to present my profile, skills, and projects in a clean and professional way.
The application runs entirely on the client side and does not require any backend services.

Key goals of this project:
- Practice **SPA architecture** without frameworks
- Apply **semantic HTML** and modern **CSS layout techniques**
- Use **JavaScript (ES6+)** for routing, rendering, and validation
- Deploy a real project using **GitHub Pages**

---

## Technologies Used

- **HTML5**
  - Semantic structure (header, nav, main, footer)
- **CSS3**
  - Flexbox & Grid
  - Responsive design
  - Custom components and layout system
- **JavaScript (ES6+)**
  - Hash-based SPA routing
  - DOM manipulation
  - Event handling
  - Form validation
- **Git & GitHub**
  - Version control with multiple commits
- **GitHub Pages**
  - Live deployment

---

## Features

- Single Page Application (SPA) using URL hash routing
- Dynamic page rendering without reload
- Responsive navigation with hamburger menu
- Projects loaded asynchronously from a local JSON file
- Client-side contact form validation
- Clean and modern UI design
- Mobile, tablet, and desktop support
